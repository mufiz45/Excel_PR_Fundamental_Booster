# Excel_PR_Fundamental_Booster
Excel PR Fundamental Booster Project

This project is a hands-on Excel workbook designed to demonstrate strong fundamentals in data analysis and formula-based problem solving. It covers real-world use cases across student performance analysis, sales reporting, and employee data management.

Key Skills Demonstrated

Conditional logic using IF, Nested IF, AND / OR

Aggregation with COUNTIFS, SUMIFS, AVERAGEIFS

Lookup techniques using XLOOKUP, INDEXâ€“MATCH

Dynamic calculations with FILTER, OFFSET, INDIRECT

Date and time analysis (YEAR, DATEDIF, TODAY)

Data formatting and structured worksheet design

Workbook Structure

Students_Data: Grade classification, performance filtering, enrollment analysis

Sales_Data: Regional sales analysis and discount eligibility logic

Employee_Data: Salary lookup and experience calculation

Purpose

Built to strengthen Excel fundamentals required for data analyst, business analyst, and operations roles, and suitable for practice, assessments, and interview preparation.
